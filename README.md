### 一、相关技术

- 前端：`thymeleaf`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示

系统分为：用户、管理员。

用户功能：1. 主页、2. 博客详情、3. 归档、4. 友链、5. 关于

管理员功能：1. 登录、2. 仪表盘、3. 发布文章、4. 文章管理、5. 页面管理、6. 评论管理、7. 标签管理、8. 文件管理、9. 友链管理、10. 系统设置
